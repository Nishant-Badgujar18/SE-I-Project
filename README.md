# Sudoku-validator/Solver

Simple Sudoku application using python to validate and Solve Sudoku.

# Installation:

1. Clone Respository

    ```makefile
    $ git clone https://github.com/Nishant-Badgujar18/Sudoku-Validator.git
    ```

# Run:

-   To Run Sudoku Tool

    ```makefile
    $ python3 Sudoku_Tool.py
    ```
